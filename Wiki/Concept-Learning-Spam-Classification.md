# Concept Learning — Spam Classification

## Overview
This section details the implementation of the **Find-S** algorithm for spam email classification.

## Key Features
- **Promotional words**
- **Suspicious links**
- **Known sender**
- **Use of capital letters**

## Algorithm
The Find-S algorithm starts with the most specific hypothesis and generalizes it as it encounters positive examples.

## Results
The learned hypothesis successfully classifies test emails based on the training data.
