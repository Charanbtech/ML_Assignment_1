# Decision Tree — Loan Default Prediction

## Overview
This section covers the construction of a decision tree for predicting loan defaults using continuous attributes.

## Attributes
- **Annual Income** (Continuous)
- **Credit Score** (Continuous)

## Validating the Model
The model uses **Information Gain** to select the best split points for continuous variables.

## Conclusion
The resulting tree provides a clear set of rules for approving or rejecting loans based on income and credit score.
