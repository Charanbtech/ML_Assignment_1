# ID3 Decision Tree — Buy Computer

## Overview
This section explains the implementation of the **ID3 algorithm** to predict computer purchases.

## Methodology
- Calculate **Entropy** of the dataset.
- Compute **Information Gain** for each attribute.
- Select the attribute with the highest gain as the root.
- Recursively build the tree.

## Attributes Analysis
- **Age**
- **Income**
- **Student Status**
- **Credit Rating**

## Prediction
The model correctly predicts whether a user buys a computer based on their demographic profile.
